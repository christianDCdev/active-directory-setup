<h1>Setup for Active Directory Infrastructure</h1>
Welcome!  This project serves as the first step to Active Directory implementation.  The objective of this project to simulate an environment in Azure that allows the implementation of on-premises Active Directory.  We will be connecting two virtual machines that each serve their own purpose.  The first virtual machine will be the Domain Controller, and it will act as a server for the second virtual machine, the Client, to connect to.
<br />
